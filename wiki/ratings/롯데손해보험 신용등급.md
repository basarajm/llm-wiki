# 롯데손해보험 신용등급

손해보험사 특성상 무보증사채·CP 등급보다 **보험금지급능력평가(IFSR, Insurer Financial Strength Rating)**가 핵심 지표입니다.

## 등급 이력

### 보험금지급능력평가(IFSR) — 주요 이력

| 평가기간 | 평가기관 | 등급 |
|---|---|---|
| 2017~2018 | 한국신용평가·한국기업평가 | A+(안정적) |
| 2019.05 | 한국신용평가·한국기업평가 | A+(부정적 검토 대상) |
| 2019.10 | 한국기업평가·한국신용평가 | A(안정적) ← A+에서 강등 |
| 2021.02~05 | 한국신용평가·한국기업평가 | A(부정적) |
| 2022.01 | 한국신용평가·한국기업평가 | A(부정적) |
| 2022.05 | 한국기업평가·한국신용평가 | A(안정적) ← 등급전망 상향 |
| 2023~2024 | 한국기업평가·한국신용평가 | A(안정적) |
| 2025.01.10 | 한국기업평가 | A(안정적) |
| 2025.05.13 | 한국기업평가 | A(부정적) ← 전망 하향 |
| 2025.11.07 | 한국기업평가 | A(부정적 검토) |
| 2026.01.09 | 한국기업평가 | A(부정적 검토) |
| **2026.03.06** | **한국기업평가** | **A-(부정적 검토)** ← 등급 강등 |

주) 보고서 기재 IFSR 이력 요약. 2025~2026년 최근 이력은 한국기업평가 기준으로 정리됨.

### 주요 경영효율지표상 평가

- 보험금지급능력평가: A(부정적 검토) [제81기 기준] → 2026.03.06 A-(부정적 검토)로 조정
- 금융소비자보호 실태평가: 제79기 '보통' (제80·81기는 자율 진단)

## 평가 근거

- **등급 추세는 하향:** 2017년 A+(안정적)에서 2019년 A로 강등된 이후 A등급을 유지했으나, 2025년 5월 전망이 '부정적'으로, 2025년 11월 '부정적 검토'로, 2026년 3월 등급 자체가 **A-**로 하향.
- **하향 배경(추정):** 자본총계 급감(1.3조→0.6조)과 K-ICS 지급여력비율 하락(2023년 213.20% → 2025년 159.48%, 경과조치 前 126.06%)에 따른 재무건전성 약화가 등급 압박 요인으로 작용한 것으로 보임(보고서상 직접 사유 미명시 — 추정).
- '부정적 검토'는 향후 추가 하향 가능성을 시사.
