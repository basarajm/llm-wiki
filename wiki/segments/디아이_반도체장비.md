## 사업 내용

디아이의 핵심 사업 부문으로, 메모리 반도체의 신뢰성을 검증하는 검사장비를 제조·공급합니다. 1988년 부설연구소 설립, 1989년 Dynamic Burn-in Test System 국산화, 1993년 일본 JEC와 기술제휴로 Monitoring Burn-In Tester 양산화에 성공한 이래 국내 번인 테스터 시장에서 경쟁우위를 유지하고 있습니다.

주요 제품군:
- **Monitoring Burn-In Tester (MBT)**: 패키지된 디바이스를 번인 보드에 장착, 고온 등 극한 환경에서 신뢰성을 검증하는 후공정 검사장비.
- **Test Burn-In Tester (TBT)**: MBT를 고도화한 메모리 최종 패키지 단계 번인 검사장비.
- **Wafer Level Burn-In Tester (WLBI) / Wafer Memory Tester (WMT)**: 웨이퍼 상태에서 대량 번인·기능 검사를 수행하는 고부가 전공정 검사장비.
- **Multi Flexibility Tester (MFT)**: 고속·광폭 I/O의 차세대 메모리 테스트 설비.
- **Logic Burn-In Tester (LBT)**: 비메모리(시스템 반도체) 특화 장비.
- **Burn-In Board (BIB)**: 디바이스를 장착해 고온 챔버에서 초기 불량을 선별하는 검사용 기판.

2025년에는 HBM용 Wafer Tester와 DDR5용 Package Tester 공급이 실적을 견인했습니다. 핵심 종속회사 ㈜디지털프론티어(지분 65.18%)가 SK하이닉스향 HBM/DDR5 테스터 공급을 담당합니다.

## 경쟁 현황

국내 반도체 장비 관련 업체는 약 100여 개사. 번인 검사장비 분야는 디아이 포함 국내 4개사 과점 구조로 진입장벽이 높습니다(고객사와의 기술협력·지속적 A/S 필요). 검사장비·핸들러 분야 주요 경쟁·인접 업체: [유니테스트], [와이씨], [네오셈], [엑시콘], [제이티], [테크윙], [한미반도체](/companies/한미반도체). 증착·전공정 장비는 [주성엔지니어링](/companies/주성엔지니어링)·원익IPS 등.

> Burn-In Board 국내 점유율: 디아이 38%(2025), 31%(2024), 36%(2023). MBT·TBT는 전방 설비투자 규모·해외 점유율 파악 곤란으로 공시 생략.
