지주회사이자 사업형 모회사로서, 밸류체인은 ① 지배하는 자회사(다운스트림 사업)와 ② 연결 사업의
원재료 조달(업스트림) 두 축으로 구성된다.

## 주요 공급사

연결 사업별 주요 원재료(보고서 II.3 원재료 현황 기준):

| 품목 | 부문 | 공급사 | 비고 |
|---|---|---|---|
| 면방·원단 | 법인영업/의류 | 티에이케이텍스타일, 대한방직 外 | 의류·산업자재 원료 |
| 식자재·농수축산물 | 푸드(현대그린푸드) | 다수 식자재 협력사 | B2B 조달 |
| 인조대리석·PVC 원료 | 건축자재(현대엘앤씨) | 석유화학·수지 공급사 | |
| 가구용 목재·자재 | 가구(현대리바트) | 목재·부자재 공급사 | |
| 건설기계 부품 | 중장비(현대에버다임) | 부품 협력사 | |

> 사업이 이종 다부문으로 분산되어 특정 단일 공급사 의존도는 낮다.

## 주요 고객사 (지배 자회사 — 다운스트림)

현대지에프홀딩스가 직접 지배하는 주요 상장 자회사(2025.12.31 직접 지분율):

| 자회사 | 지분율 | 사업 |
|---|---|---|
| [현대백화점](/companies/현대백화점) | 35.1% | 백화점·면세 |
| [현대홈쇼핑](/companies/현대홈쇼핑) | 57.4% | TV홈쇼핑 |
| [현대그린푸드](/companies/현대그린푸드) | 39.3% | 단체급식·식자재 |
| [현대이지웰](/companies/현대이지웰) | 50.0% | 복지·모바일식권 |
| [현대리바트](/companies/현대리바트) | 41.2% | 가구 |
| [현대에버다임](/companies/현대에버다임) | 45.2% | 건설기계 |
| [대원강업](/companies/대원강업) | 32.8% | 자동차 부품 |

간접 지배(종속회사 경유): [한섬](/companies/한섬)(의류)·[현대엘앤씨](/companies/현대엘앤씨)(건축자재)·
[현대바이오랜드](/companies/현대바이오랜드)(바이오)·[현대퓨처넷](/companies/현대퓨처넷)(전기통신).

최종 소비 고객은 B2C(백화점·홈쇼핑·의류·가구)와 B2B(단체급식·식자재·건축자재·건설기계)로 이원화된다.

## 협상력 분석

**공급사 협상력**: 이종 다부문 사업으로 원재료가 분산되어 단일 공급사 협상력은 제한적. 다만 식자재·
석유화학 원료·목재 등은 원자재 가격 변동에 노출.

**고객사 협상력(자회사 측면)**: 지주회사로서 자회사에 대한 지배력이 안정적(주요 자회사 지분 30~57%,
오너 일가 통한 추가 지배). 최종 소비단에서는 백화점·홈쇼핑·의류 모두 내수 소비경기에 민감해 경기
하강 시 고객(소비자) 협상력이 상대적으로 강해진다.
