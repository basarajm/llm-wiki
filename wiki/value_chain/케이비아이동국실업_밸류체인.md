# 케이비아이동국실업 밸류체인

## 주요 공급사

- 주원재료는 플라스틱 수지(사출·블로우 성형용)이며, 사업보고서에 개별 공급사가 명시되지 않아 미확인.
- 원재료 가격은 유가·석유화학 시황에 연동되어 마진 변동 요인으로 작용(추정).

## 주요 고객사

자동차 내외장재를 OEM 방식으로 완성차·모듈사에 공급한다.

| 고객사 | 비고 |
|---|---|
| [현대자동차](/companies/현대자동차) | 국내 1차 협력(완성차) |
| [기아](/companies/기아) | 국내 1차 협력(완성차) |
| [현대모비스](/companies/현대모비스) | 1차 협력(모듈) |
| 강소모비스(현대모비스 중국법인) | 중국법인(염성동국기차배건) 납품 |
| 동풍열달기아자동차 | 중국 완성차 |
| Volkswagen / Audi / BMW | 유럽 완성차(KDK Automotive GmbH 공급) |
| GM | 해외 완성차 |
| Hyundai Mobis Mexico | 멕시코법인(DONG KOOK MEXICO) 납품처 |

- 2025년 '현대모비스 파트너스 데이' 최우수 협력사 대상 수상 이력(2023.03).

## 협상력 분석

- **고객 집중도 높음**: 매출 대부분이 현대차·기아·현대모비스 그룹 OEM향으로, 완성차 가격·물량 협상력에 종속.
- **지역 다변화**: 유럽(VW·BMW·Audi)·멕시코(Hyundai Mobis Mexico)·중국으로 고객 분산 진행 중이나 여전히 현대차그룹 의존도 높음(추정).
- 신차종 부품 입찰 수주 구조로, 차종 라이프사이클에 따라 납품물량·수주총액 변동.
