---
description: 신규 페이지 타입 제안 → 승인 → taxonomy 등록
argument-hint: [타입 아이디어]
---

`engine\OPERATIONS.md` > "3. Taxonomy 관리" 절차를 따르라.

아이디어: $ARGUMENTS

1. 제안 타입을 `engine\taxonomy.md`의 `## 신규 주제 후보`에 기록한다:
   `- **타입명**: 필요한 이유 / 예상 frontmatter 필드 / 예상 섹션`
2. 사용자에게 보고하고 승인을 요청한다. **승인 전에는 타입을 직접 등록하지 않는다.**
3. 승인 시 등록 체크리스트(taxonomy 표·스키마·`wiki\` 디렉토리·`engine\templates\`·변경 이력·CLAUDE.md)를 모두 수행한다.
